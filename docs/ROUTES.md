# 讀書筆記本 (Book Notes) - 路由設計文件

本文件定義系統中所有的 URL 路徑與對應的處理邏輯，做為前後端實作的橋樑。

## 1. 路由總覽表格

| 功能 | HTTP 方法 | URL 路徑 | 對應模板 | 說明 |
| --- | --- | --- | --- | --- |
| 首頁與搜尋列表 | GET | `/` | `index.html` | 顯示所有筆記，或根據 `query` 參數顯示搜尋結果 |
| 新增筆記頁面 | GET | `/add` | `form.html` | 顯示空白的新增表單 |
| 建立筆記 | POST | `/add` | — | 接收表單資料，存入資料庫後重導向至首頁 |
| 編輯筆記頁面 | GET | `/edit/<id>` | `form.html` | 根據 ID 查詢資料，並預先填入表單中 |
| 更新筆記 | POST | `/edit/<id>` | — | 接收表單修改資料，更新資料庫後重導向至首頁 |
| 刪除筆記 | POST | `/delete/<id>` | — | 刪除指定 ID 的筆記，完成後重導向至首頁 |

## 2. 每個路由的詳細說明

### `GET /` (首頁與搜尋)
- **輸入**: URL Query 參數 `query` (非必填，例如 `/?query=哈利波特`)
- **處理邏輯**: 
  - 若無 `query` 參數，呼叫 `BookModel.get_all()`
  - 若有 `query` 參數，呼叫 `BookModel.search(query)`
- **輸出**: 將取得的陣列傳給 `index.html` 的 `books` 變數渲染。
- **錯誤處理**: 若資料庫為空或搜尋不到東西，回傳空陣列，由前端顯示「目前沒有筆記」的提示。

### `GET /add` (新增頁面)
- **輸入**: 無
- **處理邏輯**: 單純回傳表單頁面。
- **輸出**: 渲染 `form.html`。

### `POST /add` (建立筆記)
- **輸入**: 表單資料（書名、作者、心得、星級、留言）
- **處理邏輯**: 呼叫 `BookModel.create()` 寫入資料庫。
- **輸出**: 重新導向 (Redirect) 至 `/`。
- **錯誤處理**: 可在後端檢查 `title` 是否為空，若為空則重新渲染表單並提示錯誤。

### `GET /edit/<id>` (編輯頁面)
- **輸入**: URL 參數 `<id>`
- **處理邏輯**: 呼叫 `BookModel.get_by_id(id)` 取得該筆記詳細資料。
- **輸出**: 將取得的資料作為 `book` 變數傳遞給 `form.html` 渲染。
- **錯誤處理**: 若找不到該筆記資料，回傳 404 錯誤頁面。

### `POST /edit/<id>` (更新筆記)
- **輸入**: URL 參數 `<id>` 與表單修改後的資料。
- **處理邏輯**: 呼叫 `BookModel.update()` 覆寫該筆筆記。
- **輸出**: 重新導向 (Redirect) 至 `/`。

### `POST /delete/<id>` (刪除筆記)
- **輸入**: URL 參數 `<id>`
- **處理邏輯**: 呼叫 `BookModel.delete(id)` 移除該筆資料。
- **輸出**: 重新導向 (Redirect) 至 `/`。
- **注意**: 使用 HTML 的 `<form method="POST">` 送出刪除請求，不使用 GET 以防被機器人或意外點擊誤刪。

## 3. Jinja2 模板清單

所有的模板將放置於 `app/templates/` 目錄中：

1. **`base.html`**
   - 基礎版型，包含共用的 `<head>`、網站導覽列 (Navbar) 與頁尾 (Footer)。
   - 透過 `{% block content %}{% endblock %}` 讓其他頁面填入主要內容。
2. **`index.html`**
   - 繼承自 `base.html`。
   - 負責顯示搜尋框與「新增筆記」按鈕。
   - 負責用迴圈跑出每一筆書籍筆記的卡片（Card）。
3. **`form.html`**
   - 繼承自 `base.html`。
   - 將 **「新增」與「編輯」共用同一個表單**。
   - 透過判斷是否有傳入 `book` 變數來決定表單的 `action` 網址（是要 post 到 `/add` 還是 `/edit/<id>`），以及設定各欄位的 `value` 預設值。
